Demo Page for Chrome Extension

This page is only for testing your content script. You can use it to practice selecting paragraphs, spans, and sentences.

Chrome extensions can run JavaScript on normal web pages. A content script is the file that gets injected into the page. You can use it to read text, change styles, and listen for keyboard events.

This is the second paragraph. It has several sentences. Try highlighting this paragraph with your extension. Then try changing only one sentence at a time.

The Tab key should move forward through the text. Shift plus Tab should move backward. Later, the same idea can be used inside a PDF viewer.

This paragraph contains some inline text inside spans. This sentence is inside a span. Here is another span sentence. The extension should be able to find these spans too.

Building small experiments first is better than trying to build the full project immediately. Start with paragraphs. Then move to sentences. After that, connect the same logic to PDF text.
